# Reserved column names

The following column names should be used only if they adhere to the definitions in this guide:
- `year`
- `half_year`
- `quarter`
- `month`
- `week`
- `date`
- `datetime`
- `time`
- `financial_year`
- `financial_half_year`
- `financial_quarter`
- `latitude`
- `longitude`
- `x_coord`
- `y_coord`
- `postal_code`
